# Route `/updates` (désactivée)

Les fichiers ici sont la **copie** de l’ancienne app route `app/updates/`.  
Pour la réactiver : recréer `app/updates/`, y copier `layout.tsx` et `page.tsx`, et dans `next.config.js` remettre les redirections `/blog` → `/updates` si besoin.

Les textes (`t.updates` dans `lib/i18n.ts`) sont conservés dans le projet.
